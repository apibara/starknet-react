Hook to create a `ContractFactory`.

## Usage

```ts
 function Component() {
   const { contractFactory } = useContractFactory({
     compiledContract: compiledErc20,
     classHash: erc20ClassHash,
     abi: compiledErc20.abi,
   })

   return <p>Nothing to see here...</p>
 }
```

## Options

 * __compiledContract__`: CompiledContract`
    - The compiled contract.
    - CompiledContract from Starknet.js

 * __classHash__`: string`
    - The class hash.

 * __abi__`: Abi`
    - The contract abi.
    - Abi from Starknet.js

## Returns

 * __contractFactory?__`: ContractFactory`
    - The contract factory.
    - ContractFactory from Starknet.js.
