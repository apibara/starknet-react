Access the current provider, should be inside a StarknetConfig.

## Usage

```ts
import { useProvider } from "@starknet-react/core"

function Component() {
    const { provider } = useProvider()
}
```

## Returns

 * __provider__`: ProviderInterface`
    - The current provider.
    - ProviderInterface from Starknet.js.
