Get the address associated to a stark name.

## Usage

```tsx
function Component() {
  const { data, isLoading, isError } = useAddressFromStarkName({
    name: "vitalik.stark",
  });

  if (isLoading) return <span>Loading...</span>;
  if (isError) return <span>Error fetching address...</span>;
  return <span>address: {data}</span>;
}
```

## Options

- __name?__`: string`
    - Stark name.
- __contract?__`: string`
    - Naming contract to use.

## Returns

- __data?__`: string`
  - The starknet address.
