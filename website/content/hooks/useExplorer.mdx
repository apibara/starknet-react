Access the current explorer, should be inside a StarknetConfig.

## Usage

```ts
import { useExplorer } from "@starknet-react/core"

function Component() {
  const explorer = useExplorer()

  return (
    <a href={explorer.block({hash: "0x07f61cd8dedcc1978b36b2dd641b30d0b2c13190ebb356efa83e1981adf8e4ef", number: 456194})} target="_blank" rel="noreferrer">Go to {explorer.name}</a>
  )
}
```

## Returns

 * __explorer__`: Explorer`
    - The current explorer, can be sark compass, starkscan, viewblock, voyager.
